계정 삭제 요청

부비서 계정과 연결된 개인 데이터 삭제를 요청하실 수 있습니다. 앱에 로그인 가능하시다면 앱 → 설정 → 내 계정 → 회원 탈퇴 가 더 빠릅니다.

⚠️ 삭제 진행 후 영향

  • 계정 즉시 비활성화 (모든 기기 자동 로그아웃)
  • 매물·고객·일정·채팅·커뮤니티 게시글 등 개인 데이터 90일 후 영구 삭제
  • 대표(OWNER)가 요청 시 소속 직원 계정도 일괄 비활성화
  • 관계 법령상 보존 의무 데이터(계약 기록 5년, 본인확인 6개월 등)는 법정 기간 동안 보존 후 별도 파기

처리 절차

  • 아래 폼 제출 → 입력하신 이메일로 인증 메일 발송 (24시간 유효)
  • 이메일의 인증 링크 클릭 → 본인 확인 완료 → 즉시 비활성화
  • 본인이 요청하지 않은 경우 인증 링크를 무시하시면 됩니다
